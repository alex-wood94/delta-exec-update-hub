## About Delta Exec

Delta Exec is a Windows-based Lua script executor made for Roblox users who want a simple path to launch custom scripts. Its interface is kept clean and responsive so you can focus on experimenting with game changes and automation instead of fighting the tool itself. With a small install size under 40 MB, it starts fast and works smoothly with the Roblox desktop client.

Included in the app is a curated script hub that brings together more than 500 community scripts for well-known Roblox experiences. If you want to automate routine actions, adjust gameplay mechanics, or just explore Lua scripting, Delta Exec provides a practical place to begin. The built-in update system helps keep compatibility fixes current without requiring manual checks.

```bash
cd delta-exec-pc-hub
```

2. Launch the executor:

4. Use the script hub browser or paste your own Lua script into the editor and click **Execute**.

## FAQ

**Is Delta Exec safe to use on my main Roblox account?**
As with any third-party tool that connects to Roblox, there is always some risk involved. For testing scripts and modifications, an alternate account is the safer choice.

**How often does the auto-update check for new versions?**
The update engine looks for new releases once every 24 hours when the app starts. If a compatibility fix is critical, it can be pushed immediately.

**Can I use Delta Exec alongside other script executors?**
Using more than one injector at the same time can lead to conflicts or crashes. One executor at a time is the safer approach.

**Does the script hub require an internet connection?**
Yes, an active internet connection is needed to browse and download scripts from the hub. Scripts saved locally can still be used offline.

**Where are my custom scripts stored?**
Any scripts you add yourself are stored as plain text files in `Scripts/local/`. They can be backed up or shared without restriction.
